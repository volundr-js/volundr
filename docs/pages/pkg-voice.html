@volundr/voice

Audio playback - FFmpeg, Opus, AudioResource.

npm install @volundr/voice

createAudioResource

The main factory function. Takes a file path, URL, or stream and returns an AudioResource ready to play.

import { createAudioResource, StreamType } from "@volundr/voice";

// File (FFmpeg transcodes automatically)
const r1 = createAudioResource("./song.mp3");

// URL
const r2 = createAudioResource("https://example.com/audio.ogg");

// Ogg/Opus stream (skip FFmpeg)
const r3 = createAudioResource(oggStream, {
    inputType: StreamType.OggOpus,
});

// Raw Opus (zero-copy, no processing)
const r4 = createAudioResource(opusStream, {
    inputType: StreamType.Opus,
});

// With metadata
const r5 = createAudioResource("./song.mp3", {
    metadata: { title: "My Song", requestedBy: "User#1234" },
    volume: 0.8,
});
CreateAudioResourceOptions<T>
inputType StreamType - default: Arbitrary (FFmpeg)
metadata T - custom metadata attached to the resource
volume number - volume level (1.0 = normal, uses FFmpeg -af volume filter)

StreamType

Determines how the input is processed.

ValuePipelineWhen to use
StreamType.ArbitraryFFmpeg → OggDemuxer → OpusDefault. Any audio format (mp3, wav, flac, etc.)
StreamType.OggOpusOggDemuxer → OpusAlready Ogg/Opus - skips FFmpeg entirely
StreamType.Opuspass-throughRaw Opus frames - zero processing, zero-copy

String inputs (file path / URL) always use FFmpeg regardless of inputType.

Playing audio

import { createAudioResource } from "@volundr/voice";

// Get VoiceConnection from client
const conn = await guild.joinVoice("channel-id");
const player = conn.getAudioPlayer();

// Play a file
const resource = createAudioResource("./music.mp3");
player.play(resource);

// Listen for end
player.on("idle", () => {
    console.log("Playback finished");
});

getFFmpegPath

Returns the detected FFmpeg binary path. Checks ffmpeg-static first, then system PATH.

import { getFFmpegPath } from "@volundr/voice";

const path = getFFmpegPath();
console.log(path); // "/usr/bin/ffmpeg" or path from ffmpeg-static
Requirement: FFmpeg must be installed on your system, or install ffmpeg-static as an npm dependency. Völundr detects both automatically.
