@volundr/client

The main package. Everything you need in one import.

npm install @volundr/client

What's included

  • Client - connects to Discord, emits events, manages cache, declarative interaction routing
  • Entities - Guild, Message, User, Channel, Role, GuildMember, VoiceState, Presence, Interaction, etc.
  • Builders - EmbedBuilder, SlashCommandBuilder, ButtonBuilder, ModalBuilder, Components V2, etc.
  • Inline DSL - embed(), row(), button() helpers for quick message building
  • Cache - Collection, LimitedCollection, Sweepers
  • Collectors - MessageCollector, ComponentCollector, ReactionCollector (with Symbol.dispose)
  • Formatters - fmt tagged template, Formatters utility (mentions, markdown, timestamps, escape)
  • Utilities - SnowflakeUtil, Permissions, Colors, BitField, Resolvers, CDN, Locale
  • WebhookClient - standalone webhook messaging without a full Client
  • AttachmentBuilder - build file attachments from Buffer/Blob

Client - interaction routing

Register handlers declaratively. Volundr routes interactions automatically.

MethodDescription
command(name, handler)Handle slash commands. Supports "name/sub" and "name/group/sub" for subcommand routing.
autocomplete(name, handler)Handle autocomplete for a command. Same subcommand syntax.
contextMenu(name, handler)Handle user/message context menu commands.
button(match, handler)Handle button clicks. match is a string or RegExp.
selectMenu(match, handler)Handle select menu interactions. match is a string or RegExp.
modal(match, handler)Handle modal submissions. match is a string or RegExp.

Inline Message DSL

Quick helpers for building messages without full builder classes.

import { embed, field, row, button, linkButton, option } from "@volundr/client";

// Plain object embed
const e = embed({
    title: "Hello",
    description: "World",
    color: 0xFF0000,
    fields: [field("Name", "Value", true)],
});

// Quick action row
const r = row(
    button("confirm", "Confirm", ButtonStyle.Success),
    linkButton("https://example.com", "Visit"),
);

// Select option
const o = option("Choice A", "a", "Description");

InlineMessage

Callback-based fluent builder for messages.

import { InlineMessage } from "@volundr/client";

channel.send((msg) => {
    msg.content = "Hello!";
    msg.embed((e) => {
        e.title = "Title";
        e.color = 0x00FF00;
        e.field("Name", "Value");
        e.footer("Footer text");
    });
    msg.row(button("ok", "OK"));
});

Locale

import { createLocaleMap, resolveLocale } from "@volundr/client";

const t = createLocaleMap({
    now_playing: {
        "en-US": "Now Playing",
        "pt-BR": "Tocando Agora",
    },
});

t("now_playing", "pt-BR"); // "Tocando Agora"
t("now_playing", "ja");    // "Now Playing" (fallback to en-US)

Pagination

import { paginate, paginateAll } from "@volundr/client";

// Async generator - process page by page
for await (const page of paginate(
    (opts) => client.listMembers(guildId, opts),
    { limit: 100 },
)) {
    console.log(page.length); // up to 100 per page
}

// Collect everything into a flat array
const allMembers = await paginateAll(
    (opts) => client.listMembers(guildId, opts),
);
