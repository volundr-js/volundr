Sweepers

Automatically clean up old cache data to control memory usage.

Configuration

import { Client, messageLifetimeFilter } from "@volundr/client";

const client = new Client({
    token: "...",
    intents: ...,
    sweepers: {
        messages: {
            interval: 60_000,           // run every 60 seconds
            filter: messageLifetimeFilter(30 * 60_000), // remove messages older than 30 min
        },
        presences: {
            interval: 300_000,
            filter: () => true,        // clear all presences every 5 min
        },
    },
});

Built-in filters

FilterDescription
messageLifetimeFilter(ms)Remove messages older than the specified duration
lifetimeFilter(ms)Generic lifetime filter for any entity with a timestamp
() => trueRemove everything (clear the cache)

Sweepable caches

You can configure sweepers for: messages, users, presences, voiceStates, members, threads.

Tip: For large bots (100+ guilds), always configure message sweepers. Messages are the biggest memory consumer.
