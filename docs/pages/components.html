Components

Buttons, select menus, and modals.

Buttons

import { ActionRowBuilder, ButtonBuilder } from "@volundr/client";
import { ButtonStyle } from "@volundr/types";

const row = new ActionRowBuilder().addComponents(
    new ButtonBuilder().setCustomId("yes").setLabel("Confirm").setStyle(ButtonStyle.Success),
    new ButtonBuilder().setCustomId("no").setLabel("Cancel").setStyle(ButtonStyle.Danger),
);

channel.send({ content: "Are you sure?", components: [row.toJSON()] });

Or with the callback builder (zero imports):

channel.send(msg => {
    msg.content = "Are you sure?";
    msg.row(
        new ButtonBuilder().setCustomId("yes").setLabel("Confirm").setStyle(ButtonStyle.Success),
        new ButtonBuilder().setCustomId("no").setLabel("Cancel").setStyle(ButtonStyle.Danger),
    );
});

Handling button clicks

Declarative routing with client.button(). Accepts a string (exact match) or RegExp:

client.button("yes", (interaction) => {
    interaction.reply("Confirmed!");
});

client.button("no", (interaction) => {
    interaction.reply("Cancelled.");
});

// RegExp matching for dynamic IDs
client.button(/^ticket-(\d+)$/, (interaction) => {
    const match = interaction.customIdMatches(/^ticket-(\d+)$/);
    const ticketId = match?.[1];
    interaction.reply(`Opening ticket ${ticketId}`);
});

Select menus

import { StringSelectBuilder } from "@volundr/client";

const select = new StringSelectBuilder()
    .setCustomId("color")
    .setPlaceholder("Pick a color")
    .addOptions(
        { label: "Red", value: "red" },
        { label: "Blue", value: "blue" },
        { label: "Green", value: "green" },
    );

const row = new ActionRowBuilder().addComponents(select);
channel.send({ content: "Choose:", components: [row.toJSON()] });

Modals

import { ModalBuilder, ActionRowBuilder, TextInputBuilder } from "@volundr/client";
import { TextInputStyle } from "@volundr/types";

const modal = new ModalBuilder()
    .setCustomId("feedback")
    .setTitle("Feedback")
    .addComponents(
        new ActionRowBuilder().addComponents(
            new TextInputBuilder()
                .setCustomId("message")
                .setLabel("Your feedback")
                .setStyle(TextInputStyle.Paragraph)
                .setRequired(true),
        ),
    );

interaction.showModal(modal);

Handling modal submit

client.modal("feedback", (interaction) => {
    const text = interaction.getTextInputValue("message");
    interaction.reply(`Thanks for the feedback: ${text}`);
});

Handling select menus

client.selectMenu("color", (interaction) => {
    const selected = interaction.values[0];
    interaction.reply(`You picked: ${selected}`);
});
Pattern matching: All component handlers (.button(), .selectMenu(), .modal()) accept a string for exact match or a RegExp for pattern matching on the custom_id.
