@volundr/types

TypeScript type definitions for the Discord API.

npm install @volundr/types

Pure type package - no runtime code, no dependencies. Used internally by all @volundr/* packages.

import type { APIMessage, APIUser, APIGuild, ChannelType } from "@volundr/types";
