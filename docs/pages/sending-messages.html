Sending Messages

Three ways to send. Same result. Works everywhere.

The golden rule

Every .send() and .reply() in Völundr accepts the same three input types:

1. String

channel.send("Hello!");

2. Options object

channel.send({
    content: "Hello!",
    embeds: [embed.toJSON()],
});

3. Callback builder

channel.send(msg => {
    msg.content = "Hello!";
    msg.embed(e => {
        e.title = "Welcome";
        e.description = "This is an embed";
        e.color = 0x5865F2;
        e.field("Users", "1,234", true);
        e.field("Servers", "56", true);
    });
    msg.row(
        new ButtonBuilder().setCustomId("ok").setLabel("Got it").setStyle(ButtonStyle.Primary),
    );
});
Tip: The callback builder requires zero imports. Build embeds, buttons, and action rows inline.

Works on every entity

The same MessageInput type is accepted by:

  • TextChannel.send()
  • DMChannel.send()
  • VoiceChannel.send()
  • ThreadChannel.send()
  • User.send()
  • Message.reply()

Embeds

import { EmbedBuilder, Colors } from "@volundr/client";

const embed = new EmbedBuilder()
    .setTitle("Server Info")
    .setColor(Colors.Blurple)
    .addFields(
        { name: "Members", value: "1,234", inline: true },
        { name: "Channels", value: "56", inline: true },
    )
    .setFooter({ text: "Powered by Völundr" })
    .setTimestamp();

channel.send({ embeds: [embed.toJSON()] });

Replying to messages

client.on("MESSAGE_CREATE", (message) => {
    // Simple reply
    message.reply("Got it!");

    // Rich reply
    message.reply(msg => {
        msg.embed(e => {
            e.title = "Reply";
            e.description = "This is a reply with an embed";
        });
    });
});

Tagged template mentions

Use fmt to auto-resolve entities to Discord mention syntax:

import { fmt } from "@volundr/client";

// User, Channel, Role - all auto-resolved
channel.send(fmt`Welcome ${member}! Check out ${rulesChannel}.`);

// Dates become Discord timestamps
channel.send(fmt`Event starts ${new Date("2025-01-01")}`);

Attachments

import { AttachmentBuilder } from "@volundr/client";

const file = new AttachmentBuilder("./image.png", "screenshot.png");

channel.send({
    content: "Here's the image",
    files: [file],
});
