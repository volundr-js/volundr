Creating a Bot

Set up your project and connect to Discord.

Prerequisites

  • Node.js 18+
  • A Discord bot token - create one at discord.com/developers
  • TypeScript 5+ (recommended)

Project setup

mkdir my-bot && cd my-bot
npm init -y
npm install @volundr/client
npm install -D typescript tsx

tsconfig.json

{
  "compilerOptions": {
    "target": "ES2022",
    "module": "Node16",
    "moduleResolution": "Node16",
    "strict": true,
    "outDir": "dist"
  },
  "include": ["src"]
}

Bot token

Store your token in environment variables. Never commit it to source control.

# .env
DISCORD_TOKEN=your-bot-token-here
Warning: If your token is ever exposed, regenerate it immediately in the Discord Developer Portal.

Client options

import { Client } from "@volundr/client";
import { GatewayIntents } from "@volundr/gateway";

const client = new Client({
    token: process.env.DISCORD_TOKEN!,
    intents:
        GatewayIntents.Guilds |
        GatewayIntents.GuildMessages |
        GatewayIntents.MessageContent |
        GatewayIntents.GuildMembers,
});

Intents

Intents control which events your bot receives. Only enable what you need.

IntentEvents
GuildsGuild create/update/delete, channel changes, role changes
GuildMessagesMessage create/update/delete in guild channels
MessageContentAccess to message.content (privileged)
GuildMembersMember add/update/remove (privileged)
GuildVoiceStatesVoice state updates (required for voice)
DirectMessagesDM messages
GuildPresencesPresence updates (privileged)

Run your bot

npx tsx src/index.ts
