@volundr/lavalink

Lavalink v4 client - search, play, filters, events.

npm install @volundr/lavalink

Full Lavalink v4 integration with automatic node balancing, session resume, and player management.

Quick start

import { Client } from "@volundr/client";

const client = new Client({ token: "...", intents: 3276799 });

client.lavalink.addNode({
    name: "main",
    host: "localhost",
    port: 2333,
    password: "youshallnotpass",
});

await client.login();
client.lavalink.connectAll();

// Play a track
const player = client.lavalink.createPlayer(guildId);
await player.connect(voiceChannelId, { selfDeaf: true });

const result = await player.search("ytsearch:never gonna give you up");
if (result.loadType === "search") {
    await player.play(result.data[0]);
}
