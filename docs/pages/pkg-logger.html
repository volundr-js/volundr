@volundr/logger

Structured logging with lazy evaluation.

npm install @volundr/logger

Usage

import { Logger, LogLevel } from "@volundr/logger";

// Set global level
Logger.setLevel(LogLevel.DEBUG);

// Get a logger instance (cached)
const log = Logger.getLogger("gateway", "Shard-0");

log.info("Connected");
log.debug(() => `Heartbeat ACK: ${latency}ms`); // lazy - only evaluated if DEBUG enabled
log.warn("Rate limited", { route, retryAfter });
log.error("Fatal", err);

Logger

Static Methods

MethodReturnsDescription
Logger.getLogger(namespace, name)LoggerGet or create logger (cached globally)
Logger.setLevel(level)voidSet global log level
Logger.getLevel()LogLevelGet current global level

Instance Methods

MethodDescription
trace(message, ...args)Log at TRACE level
debug(message, ...args)Log at DEBUG level
info(message, ...args)Log at INFO level
warn(message, ...args)Log at WARN level
error(message, ...args)Log at ERROR level
isTraceEnabled()Check if TRACE is enabled
isDebugEnabled()Check if DEBUG is enabled

message accepts string or () => string (lazy - defers string interpolation until actually logging).

LogLevel

LevelValue

LogLevel.DEBUG1
LogLevel.INFO2
LogLevel.WARN3
LogLevel.ERROR4

Default: LogLevel.INFO. Messages are only emitted when level >= globalLevel.
