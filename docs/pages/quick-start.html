Quick Start

From zero to a working bot in under a minute.

Install

npm install @volundr/client

The simplest bot

import { Client } from "@volundr/client";
import { GatewayIntents } from "@volundr/gateway";

const client = new Client({
    token: process.env.DISCORD_TOKEN!,
    intents: GatewayIntents.Guilds,
});

client.command("ping", i => i.reply("Pong!"));
client.connect();

4 lines of actual logic. No boilerplate, no if/switch, no 47 imports.

A real bot in 30 lines

import { Client, fmt, embed, row, button } from "@volundr/client";
import { GatewayIntents } from "@volundr/gateway";
import { ButtonStyle } from "@volundr/types";

const client = new Client({
    token: process.env.DISCORD_TOKEN!,
    intents: GatewayIntents.Guilds,
});

// Slash command with embed + button
client.command("userinfo", async (i) => {
    const user = i.options.getUser("target", i.client) ?? i.user;
    await i.reply({
        embeds: [embed({ title: user.displayName, thumbnail: user.displayAvatarURL(), color: 0x5865F2 })],
        components: [row(button("refresh", "Refresh", ButtonStyle.Primary))],
    });
});

// Button handler with regex matching
client.button("refresh", async (i) => {
    await i.update(fmt`Refreshed by ${i.user}!`);
});

// Autocomplete
client.autocomplete("search", async (i) => {
    const query = i.focused?.value ?? "";
    await i.respond([{ name: `Result: ${query}`, value: String(query) }]);
});

client.connect();

Commands, buttons, autocomplete, embeds, tagged templates - all in 30 lines. No frameworks, no file-based routing, no decorators. Just TypeScript.

Tip: Every .command(), .button(), .selectMenu(), and .modal() returns this, so you can chain them. RegExp matching on custom_id lets you route ticket-123, ticket-456 etc. with a single handler.
