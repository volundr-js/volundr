Events

Listen to Discord events with type-safe handlers.

Listening to events

client.on("MESSAGE_CREATE", (message) => {
    console.log(message.content);
});

client.on("GUILD_MEMBER_ADD", (member) => {
    console.log(`${member.user.username} joined ${member.guild.name}`);
});

client.on("ready", () => {
    console.log("Bot is online!");
});

Event names

Völundr uses Discord's native event names in SCREAMING_SNAKE_CASE. No translation layer. You can use string literals or the GatewayEvent enum:

import { GatewayEvent } from "@volundr/types";

// Both are equivalent
client.on("MESSAGE_CREATE", (message) => { /* ... */ });
client.on(GatewayEvent.MessageCreate, (message) => { /* ... */ });

Rich entities

5 events emit rich entity objects with methods. All others emit raw Discord API data.

EventPayloadDescription
MESSAGE_CREATEMessageRich entity with .reply(), .delete(), .react()
MESSAGE_UPDATEMessageRich entity (only if cached)
INTERACTION_CREATEInteractionRich entity with .reply(), .deferReply(), etc.
GUILD_CREATEGuildRich entity (only if cached)
GUILD_MEMBER_ADDGuildMemberRich entity (only if cached)
 
client.on("MESSAGE_CREATE", (message) => {
    // message is a Message entity, not raw API data
    message.author.username;    // User entity
    message.guild;              // Guild entity (or null in DMs)
    message.channel;            // Channel entity
    message.reply("Hello!");    // method - calls REST API
});

Once

Listen for an event only once:

client.once("ready", () => {
    console.log("First ready!");
});

Event streaming

Consume events as an async iterable with client.stream(). Cancel with an AbortSignal:

// Process messages as a stream
const ac = new AbortController();

for await (const message of client.stream("MESSAGE_CREATE", { signal: ac.signal })) {
    console.log(message.content);
    if (message.content === "stop") ac.abort();
}

Useful for sequential workflows where you need to wait for a specific event:

// Wait for a specific user's message
async function waitForResponse(channelId: string, userId: string): Promise {
    for await (const msg of client.stream("MESSAGE_CREATE")) {
        if (msg.channel?.id === channelId && msg.author.id === userId) {
            return msg.content;
        }
    }
    return "";
}
