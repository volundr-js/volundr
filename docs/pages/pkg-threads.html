@volundr/threads

Worker thread pool for multi-core sharding.

npm install @volundr/threads

Run CPU-intensive work across worker threads. Used by @volundr/gateway for multi-core sharding.

Usage

import { ThreadPool } from "@volundr/threads";

const pool = new ThreadPool({ size: 4 });

// Run a function in a worker
const result = await pool.run((a, b) => a + b, 2, 3);
console.log(result); // 5

// Wait for all tasks
await pool.drain();

// Shut down
await pool.close();

ThreadPool

new ThreadPool(options?)

ThreadPoolOptions
size number - number of workers (default: os.availableParallelism())
taskTimeout number - timeout per task in ms
