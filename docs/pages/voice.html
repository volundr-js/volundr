Voice

Connect to voice channels and play audio.

Connect

const connection = client.joinVoiceChannel(guildId, channelId, {
    selfDeaf: true,
});

connection.on("ready", () => {
    console.log("Connected to voice!");
});

Play audio

import { createAudioResource } from "@volundr/voice";

// From file
const resource = createAudioResource("./song.mp3", { volume: 0.8 });
connection.getAudioPlayer()!.play(resource);

// From URL
const resource2 = createAudioResource("https://example.com/song.ogg");

// Raw Opus stream (zero-copy)
const resource3 = createAudioResource(opusStream, {
    inputType: StreamType.Opus,
});

Stream types

TypePipeline
ArbitraryFFmpeg → OggDemuxer → Opus (default)
OggOpusOggDemuxer → Opus (skip FFmpeg)
OpusPass-through (zero-copy)

Lavalink

import { LavalinkManager } from "@volundr/lavalink";

const lavalink = new LavalinkManager(client, {
    nodes: [{ host: "localhost", port: 2333, password: "youshallnotpass" }],
});

await lavalink.connect();

const result = await lavalink.search("never gonna give you up");
const player = lavalink.getPlayer(guildId);
await player.play(result.tracks[0]);
Tip: For production music bots, use @volundr/lavalink instead of native audio. It offloads encoding to a dedicated server.
