@volundr/gateway

Gateway v10 - WebSocket, compression, sharding, voice.

npm install @volundr/gateway

You usually don't import this directly - @volundr/client handles it. Use this package when you want raw gateway access without the entity layer.

Standalone usage

import { Gateway, GatewayIntents } from "@volundr/gateway";

const gateway = new Gateway({
    token: "...",
    intents: GatewayIntents.Guilds | GatewayIntents.GuildMessages,
});

gateway.on("MESSAGE_CREATE", (data) => {
    // data is raw APIMessage, not an entity
    console.log(data.content);
});

await gateway.connect();

Voice Classes

VoiceConnection

Manages the full lifecycle of a voice channel connection.

MethodDescription
joinChannel(channelId, selfMute?, selfDeaf?)Join a voice channel
leaveChannel()Leave current channel
disconnect()Full disconnect + cleanup
getStatus()Returns "disconnected" | "connecting" | "connected" | "ready"
getAudioPlayer()Returns AudioPlayer | null (available when ready)

Events: status, ready, close, error

AudioPlayer

Plays Opus audio streams over RTP with encryption.

MethodDescription
play(input)Play a Readable or { opusStream: Readable }
stop()Stop playback, send silence frames
pause()Pause playback
resume()Resume from paused
getStatus()"idle" | "playing" | "paused"
getFrameCount()Total frames sent since last play()

Events: status, error, idle

OggDemuxer

Transform stream: Ogg/Opus container → individual Opus frames.

import { OggDemuxer } from "@volundr/gateway";

fs.createReadStream("audio.opus")
    .pipe(new OggDemuxer())
    .on("data", (frame) => { /* Opus frame */ });
